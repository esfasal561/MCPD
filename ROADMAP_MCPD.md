# MCPD Technical Roadmap

## Phase 1 – Concept Consolidation
- Finalize system architecture description
- Define subsystem interfaces (MPI, PCC, PHA, CPCS, MNA)
- Establish performance envelope (Isp, thrust, Δv)

## Phase 2 – Analytical Modeling
- Magnetic intake capture efficiency model
- Plasma confinement stability analysis (MHD)
- RF heating efficiency curves (ECRH/ICRH)
- Theta‑pinch compression modeling
- Magnetic nozzle detachment simulation

## Phase 3 – Simulation Framework
- Develop simplified 2D MHD simulation
- Implement plasma particle tracking
- Model magnetic field topology
- Validate confinement and exhaust behavior

## Phase 4 – Prototype Components (bench‑scale)
- Build small‑scale magnetic intake coil
- Construct toroidal confinement test chamber
- RF heating testbed (low‑power)
- Magnetic nozzle field‑line mapping

## Phase 5 – Integrated Demonstrator
- Assemble a laboratory‑scale MCPD demonstrator
- Validate plasma capture → heating → exhaust chain
- Measure thrust and Isp in controlled environment

## Phase 6 – Mission Concept Studies
- Earth–Mars transfer optimization
- Deep‑space probe applications
- Power system integration (nuclear + electric)

## Phase 7 – Publication & Review
- White paper v2.0
- Peer review cycle
- Public technical release
